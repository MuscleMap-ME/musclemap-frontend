### Feed Content Types

| Type | Appears When |
|------|--------------|

| Streak | Someone hits a streak milestone |

### Privacy Controls

You control what appears in the feed:

```

=====================

  - Public: Everyone sees
  - Friends: Only friends see
  - Private: Nobody sees

Share these events:
[x] Workout completions
[x] Level ups
[ ] Personal records (off - private)
[x] Achievements
[ ] Competition results
```

## High Fives

### Giving High Fives

When you see something awesome, give a high five!

```
+------------------------------------------------------------+
| @FitnessFiona completed a workout!                          |
| Leg Day - 68 TU - 45 min                                   |
|                                                             |
|          [Give High Five]                                  |
|                                                             |
| 15 people gave high fives                                  |
+------------------------------------------------------------+
```

### High Five Types

| Type | Cost | Effect |
|------|------|--------|
| Standard | Free | Shows support |
| Super High Five | 5 credits | Extra celebration |
| Mega High Five | 25 credits | Animated effect + notification |

### Receiving High Fives

```
NOTIFICATIONS
=============

@IronMike gave you a high five!
  "Great workout today!"
  [High Five Back]

@FitQueen gave you a SUPER high five!
  For your new bench press PR
  [View Workout]
```

## Crews

### What Are Crews?

Crews are small teams (2-10 people) who train together.

```
+------------------------------------------------------------------+
|                       MY CREW                                     |
|------------------------------------------------------------------|
|                                                                   |
|  IRON WARRIORS                                                    |
|  "Lifting heavy, growing strong"                                  |
|                                                                   |
|  MEMBERS (5/10)                                                   |
|  +------------------------------------------------------------+  |
|  | @YourName (Captain)     | 156 TU this week | Active today   |  |
|  | @IronMike               | 203 TU this week | Active 2h ago  |  |
|  | @FitFiona               | 178 TU this week | Active today   |  |
|  | @GymGuru                | 145 TU this week | Active 1d ago  |  |
|  | @PumpKing               | 189 TU this week | Active today   |  |
|  +------------------------------------------------------------+  |
|                                                                   |
|  CREW STATS                                                       |
|  Total TU This Week: 871 TU                                       |
|  Crew Rank: #45 globally                                          |
|                                                                   |
|  [Crew Chat]  [Invite]  [Settings]                                |
|                                                                   |
+------------------------------------------------------------------+
```

### Crew Features

| Feature | Description |
|---------|-------------|
| Crew Chat | Private group messaging |
| Crew Leaderboard | Internal rankings |
| Crew Challenges | Group goals |
| Crew Wars | Compete against other crews |

### Creating/Joining Crews

**To Create:**
1. Go to Community → Crews
2. Click "Create Crew"
3. Set name, description, privacy
4. Invite friends

**To Join:**
1. Search for crews
2. Request to join
3. Wait for captain approval

## Rivalries

### 1v1 Competition

Challenge someone to a friendly rivalry:

```
+------------------------------------------------------------------+
|                     RIVALRY                                       |
|------------------------------------------------------------------|
|                                                                   |
|  @YourName  VS  @CompetitorCarl                                   |
|                                                                   |
|  THIS WEEK'S SCORE                                                |
|                                                                   |
|       156 TU                    143 TU                            |
|       [====|---]                                                  |
|                                                                   |
|  You're winning by 13 TU!                                         |
|                                                                   |
|  Duration: 2 weeks remaining                                      |
|  Stakes: Bragging rights + 50 credits                             |
|                                                                   |
|  [Send Message]  [View Profile]  [End Rivalry]                    |
|                                                                   |
+------------------------------------------------------------------+
```

### Rivalry Rules

- Both parties must accept
- Default duration: 1 week
- Winner determined by TU earned
- Optional credit stakes
- Can have up to 3 active rivalries

### Starting a Rivalry

1. Visit someone's profile
2. Click "Challenge to Rivalry"
3. Set duration and stakes
4. Wait for acceptance

## Competitions

### Joining Competitions

```
+------------------------------------------------------------------+
|                  WINTER CHALLENGE                                 |
|                  January 1-31, 2026                               |
|------------------------------------------------------------------|
|                                                                   |
|  GOAL: Earn the most TU this month                                |
|  PARTICIPANTS: 1,245                                              |
|  PRIZES: Top 10 get exclusive badge                               |
|                                                                   |
|  YOUR RANK: #156                                                  |
|  YOUR TU: 520 TU                                                  |
|  LEADER TU: 2,345 TU                                              |
|                                                                   |
|  TIME REMAINING: 16 days                                          |
|                                                                   |
|  [View Full Leaderboard]  [Share Progress]                        |
|                                                                   |
+------------------------------------------------------------------+
```

### Competition Types

| Type | Duration | Goal |
|------|----------|------|
| Weekly Sprint | 7 days | Most TU |
| Monthly Marathon | 30 days | Most TU |
| Muscle Focus | Varies | Most TU for specific muscles |
| Streak Challenge | Open-ended | Longest streak |
| PR Pursuit | Varies | Most new PRs |

### Creating Competitions

Level 6+ users can create custom competitions:

1. Set name and description
2. Choose duration
3. Set entry requirements
4. Define prizes (optional)
5. Invite or make public
