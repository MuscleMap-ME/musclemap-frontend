# Creating and Managing Crews

Build your fitness squad and train together.

---

## What is a Crew?

A Crew is a small group of users who train together, compete as a team, and support each other. Think of it as your personal fitness squad.

---

## Creating a Crew

### Step 1: Initiate

Go to **Crews > Create New Crew**

### Step 2: Configure

```
┌────────────────────────────────────┐
│         CREATE YOUR CREW           │
├────────────────────────────────────┤
│ Crew Name: [Dawn Patrol         ]  │
│ (Cannot be changed later)          │
├────────────────────────────────────┤
│ Size: 4 ── ● ──────────────── 20   │
│       Current: 8 members max       │
├────────────────────────────────────┤
│ Privacy:                           │
│ ○ Open (anyone can join)           │
│ ● Request (approval needed)        │
│ ○ Invite Only                      │
├────────────────────────────────────┤
│ Description:                       │
│ ┌────────────────────────────────┐ │
│ │ Early morning lifters who      │ │
│ │ believe the best workout is    │ │
│ │ the one done before sunrise.   │ │
│ └────────────────────────────────┘ │
├────────────────────────────────────┤
│ Requirements (optional):           │
│ ☐ Minimum Level: [  ]              │
│ ☐ Specific Archetype: [Select]     │
│ ☐ Weekly Activity: [ ] workouts    │
├────────────────────────────────────┤
│          [CREATE CREW]             │
└────────────────────────────────────┘
```

### Step 3: Invite Members

- Share crew link
- Send direct invites
- Post in your Hangout

### Step 4: Lead Your Crew

As Captain, you can:
- Set weekly challenges
- Promote officers
- Remove inactive members
- Manage crew settings

---

## Crew Roles

| Role | Permissions |
|------|-------------|
| **Captain** | Full control, can delete crew |
| **Officer** | Can approve members, set challenges |
| **Member** | Participate in challenges, chat |

---

## Crew Features

### Team Challenges
- Set collective goals (e.g., "1000 TU as a team this week")
- Everyone contributes
- Rewards for completion

### Crew Chat
- Dedicated group chat
- Share workouts
- Coordinate training times

### Crew Stats
- Combined TU leaderboard
- Member activity tracker
- Crew vs Crew competitions

---

## Best Practices

1. **Keep it active** - Set regular challenges to maintain engagement
2. **Be inclusive** - Welcome new members warmly
3. **Celebrate wins** - Recognize member achievements
4. **Communicate** - Use crew chat to build community
