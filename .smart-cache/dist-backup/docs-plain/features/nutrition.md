## Food Logging

### Adding Foods

Multiple ways to log what you eat:

```
LOGGING METHODS
===============

1. BARCODE SCAN     Scan packaged foods
2. SEARCH           Search food database
3. RECENT           Quick-add recent items
4. FAVORITES        Save frequently eaten foods
5. CUSTOM           Create custom foods/recipes
```

### Barcode Scanner

1. Tap **+** → **Scan Barcode**
2. Point camera at product barcode
3. Food appears with nutrition info
4. Adjust serving size
5. Add to meal

```

```

### Pre/Post Workout

```
WORKOUT NUTRITION TIMING
========================

PRE-WORKOUT (2-3 hours before)
├── Moderate protein (20-30g)
├── Complex carbs (40-60g)
└── Low fat (easier digestion)

INTRA-WORKOUT (during, if 90+ min)
├── Fast carbs (sports drink)
└── BCAAs (optional)

POST-WORKOUT (within 2 hours)
├── Fast protein (30-40g)
├── Fast carbs (40-80g)
└── Moderate fat okay
```

## FAQ

### How accurate is the barcode scanner?

Very accurate for major brands. Data comes from manufacturer information and is regularly updated.

### Can I import from other apps?

Currently supported:
- MyFitnessPal (CSV export)
- Cronometer (CSV export)
- Apple Health (sync)

### Do I need to track nutrition to use MuscleMap?

No. Nutrition tracking is optional. You can use all workout features without logging food.

### How do I track eating out?

Options:
- Search restaurant name
- Use generic entries
- Estimate from similar foods
- Log just macros if unsure

### What about alcohol?

Alcohol is tracked as its own category:
- 7 calories per gram
- Logged separately from macros
- Shows in daily summary
