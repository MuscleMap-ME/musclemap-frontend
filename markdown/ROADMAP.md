# MuscleMap Product Roadmap

## Core Philosophy
- **Visual-first**: Immersive 3D experiences, not spreadsheets
- **Touch-first**: Designed for mobile, works everywhere  
- **Body-positive**: Inclusive for all abilities and body types
- **Real-time**: Geo-located, data-rich, always current
- **Rewarding**: Early and continuous rewards for progress

## User Experience Modes
1. **Guided Mode** - Maximum hand-holding, step-by-step
2. **Standard Mode** - Balanced guidance
3. **Power User Mode** - Minimal UI, data-dense dashboard
4. **Zen Mode** - No numbers, just visual progress

## Navigation Paradigms
- **Goal View**: "I want to run faster" / "I want bigger arms"
- **Archetype View**: "Train like a judoka" / "Build like a bodybuilder"
- Seamless switching between views
- Cross-archetype transfer visualization

## Reward System (Immediate to Advanced)

### Day 1 Rewards (signup)
- Midnight Blue theme
- Basic Circle frame
- Confetti celebration
- Streak Counter widget
- Daily Quote widget

### Early Progress (first week)
- Forest Green theme (first workout)
- 3D Muscle View widget
- Bronze Ring frame (10 TU)

### Building Momentum
- Sunset Orange theme (3-day streak)
- Workout Calendar widget
- Silver Ring frame (25 TU)
- Fireworks celebration

### Intermediate
- Phoenix Fire theme (7-day streak)
- Zone Leaderboard widget
- Flame Border frame (14-day streak)
- Gold Ring frame (50 TU)

### Advanced
- Arctic Ice theme (50 TU)
- Champion Gold theme (archetype complete)
- Platinum Crown frame (100 TU)
- Energy Explosion celebration

## Dashboard Customization
- Drag-and-drop widget placement
- Theme backgrounds (unlockable)
- Avatar frames (unlockable)
- Celebration animations (unlockable)
- Widget skins and styles

## Community Features

### Local
- Geo-located training zones
- "People training near you" (opt-in privacy)
- Local leaderboards
- Zone challenges

### Global
- Weekly/monthly events
- Archetype championships
- Team battles (crews)
- Rivalry system with stakes

### Social
- High-five encouragement system
- Inspiration chains
- Mentor matching
- Achievement sharing

## Accessibility & Inclusion
- Disability-aware exercise substitutions
- Seated alternatives for all exercises
- Adjustable text size/contrast
- Screen reader support
- Reduced motion option
- Adaptive difficulty
- Physical limitation profiles (injuries, chronic conditions, mobility restrictions)
- Age-appropriate exercise modifications
- Personalized workout plans based on individual capabilities

## Coming Soon: 3D Anatomical Models
- **Male and female anatomical models** with individually rendered muscles
- Real-time muscle activation visualization during exercise selection
- Interactive muscle exploration to understand anatomy
- Progress visualization showing muscle development over time
- Guided workout navigation with visual muscle targeting
- Educational mode explaining muscle function and connections

## Goal-Based Training (Planned)
- Weight loss goals (5 lbs, 10 lbs, custom targets)
- Strength milestones (bench press PR, deadlift goals)
- Body composition targets
- Endurance objectives (run a 5K, improve stamina)
- Flexibility and mobility goals
- Timeline-based goal tracking with adaptive plans

## Institutional & Occupational Archetypes (Planned)
- **First Responders**: Firefighter, Police Officer, EMT/Paramedic
- **Military Branches**: Army, Navy, Air Force, Marines, Coast Guard, Space Force
- **Athletic Archetypes**: Powerlifter, Bodybuilder, CrossFitter, Judoka, Boxer, Wrestler
- **Functional Fitness**: Construction Worker, Healthcare Worker, Desk Athlete
- Each archetype includes occupation-specific fitness requirements and PT test preparation

## Data Integrations (Planned)
- Weather API (outdoor suggestions)
- Wearables (Garmin, Fitbit, Apple Watch, Whoop)
- Nutrition tracking
- Music BPM matching
- Exercise video tutorials

## Credit Economy
- 1 credit = 1% progress = real work done
- Switching archetypes: 10 credits
- Joining challenges: 5 credits
- Creating challenges: 20 credits
- Credits earned through exercise only (integrity)

## Tech Stack
- Frontend: React + Vite + Tailwind + Three.js
- Backend: Node.js + Express + SQLite
- 3D: Three.js / React Three Fiber
- Mobile: React Native (planned)
- Wearables: Native SDKs (planned)
