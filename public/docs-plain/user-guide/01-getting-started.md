## Create Account

### Sign Up Process

1. Visit [musclemap.me](https://musclemap.me)
2. Click **Sign Up**
3. Enter your email and password
4. Verify your email (optional but recommended)

```
+----------------------------------+
|         CREATE ACCOUNT           |
|----------------------------------|
|                                  |
|  Email:                          |
|  [____________________________]  |
|                                  |
|  Password:                       |
|  [____________________________]  |
|  (min 8 characters)              |
|                                  |
|  [    Create Account    ]        |
|                                  |
|  Already have an account?        |
|  [Log in]                        |
|                                  |
+----------------------------------+
```

### What You Get

| Benefit | Details |
|---------|---------|
| 100 Credits | 4 free workouts |
| Full Access | All features available |
| Progress Tracking | Your data saved forever |
| Community Access | Leaderboards, messaging |

### Archetype Details

| Archetype | Levels | Focus Areas | Best Equipment |
|-----------|--------|-------------|----------------|
| Bodybuilder | 10 | Chest, Arms, Shoulders | Full gym |
| Powerlifter | 10 | Squat, Bench, Deadlift | Barbell |
| Gymnast | 10 | Skills, Holds, Mobility | Bodyweight |
| CrossFit | 10 | All muscle groups | Varied |
| Martial Artist | 10 | Core, Legs, Power | Mixed |
| Runner | 10 | Legs, Endurance | Minimal |
| Climber | 10 | Back, Grip, Core | Pull-up bar |
| Strongman | 10 | Full body, Grip | Heavy objects |
| Functional | 10 | Balanced | Any |
| Swimmer | 10 | Shoulders, Back, Core | Minimal |

### Choosing Tips

- **New to fitness?** → Functional or Bodybuilder
- **Specific sport?** → Match your sport
- **Limited equipment?** → Gymnast or Functional
- **Strength goals?** → Powerlifter or Strongman

**Note**: You can change your archetype anytime in Settings. Your TU history is preserved.

## Set Your Equipment

MuscleMap filters exercises based on what you have.

### Equipment Options

```
EQUIPMENT SETUP
===============

Select all that apply:

[ ] Bodyweight Only
    No equipment needed. Always available.

[ ] Pull-up Bar
    Enables pull-ups, hanging exercises.

[ ] Resistance Bands
    Adds band-assisted and band-resisted moves.

[ ] Kettlebells
    Single or double KB workouts.

[ ] Dumbbells
    Classic free weight exercises.

[ ] Barbell & Plates
    Olympic lifts, powerlifting moves.

[ ] Cable Machine
    Gym cable exercises.

[ ] Full Gym Access
    All machines and equipment.
```

### Location-Based Equipment

MuscleMap also knows where you're training:

| Location | Available Equipment |
|----------|---------------------|
| Gym | Everything selected + machines |
| Home | Your home equipment |
| Park | Bodyweight + outdoor-friendly |
| Hotel | Minimal bodyweight |
| Office | Quick, quiet exercises |
| Travel | Zero equipment |

Set your location before each workout for best recommendations.

## Complete Your Profile

### Basic Information

| Field | Purpose |
|-------|---------|
| Display Name | Shown on leaderboards |
| Avatar | Your profile picture |
| Bio | Optional description |
| Location | General area (privacy-protected) |

### Physical Stats (Optional)

These improve workout recommendations:

| Stat | Why It Helps |
|------|--------------|
| Height | Exercise scaling |
| Weight | Load recommendations |
| Age | Recovery adjustments |
| Experience | Exercise complexity |

### Privacy Note

- All physical stats are private by default
- You control what appears on your profile
- Community features work without personal data
