### Minimalist Mode

For maximum privacy:

```
MINIMALIST MODE
===============

[ ] Enable Minimalist Mode

When enabled:
- You won't appear on leaderboards
- Your activity won't show in feeds
- Others can't see your profile
- You can still see everything

Note: You can still compete in private crews
and rivalries with accepted friends.

[Enable]
```

### Delete Account

```
DELETE ACCOUNT
==============

WARNING: This action is permanent!

Deleting your account will:
- Remove all your workout history
- Delete your profile and settings
- Remove you from all crews and rivalries
- Forfeit any remaining credits

This CANNOT be undone.

To confirm, type "DELETE" below:
[____________]

[Cancel]  [Delete My Account]
```

### Data Retention

| Data Type | Retention |
|-----------|-----------|
| Workout logs | Forever (until deleted) |
| Messages | 1 year after last activity |
| Account info | While account active |
| Analytics | 90 days (anonymized) |
