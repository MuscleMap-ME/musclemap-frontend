# MuscleMap Quick Start

> Get started with MuscleMap in 5 minutes.

---

## TL;DR

```
1. Sign up at musclemap.me
2. Choose your archetype (training style)
3. Set your equipment
4. Start your first workout
5. Watch your muscles light up!
```

---

## Step 1: Create Your Account

Visit [musclemap.me](https://musclemap.me) and click **Sign Up**.

```
+----------------------------------+
|         CREATE ACCOUNT           |
|----------------------------------|
|  Email:    [________________]    |
|  Password: [________________]    |
|                                  |
|  [    Create Account    ]        |
+----------------------------------+

You'll receive 100 free credits to get started!
```

---

## Step 2: Choose Your Archetype

Your archetype defines your training philosophy. Pick the one that matches your goals:

| Archetype | Best For | Style |
|-----------|----------|-------|
| Bodybuilder | Muscle aesthetics | Hypertrophy, pump |
| Powerlifter | Maximum strength | Heavy compound lifts |
| Gymnast | Bodyweight mastery | Calisthenics, skills |
| CrossFit | All-around fitness | High intensity, varied |
| Martial Artist | Combat readiness | Power, explosiveness |
| Runner | Endurance | Leg strength, cardio |
| Climber | Grip strength | Upper body, fingers |
| Strongman | Functional power | Odd objects, carries |
| Functional | General fitness | Balanced, practical |
| Swimmer | Aquatic performance | Full body, endurance |

**Tip**: You can change your archetype later in Settings.

---

## Step 3: Set Your Equipment

Tell MuscleMap what you have access to:

```
[ ] Bodyweight Only      - Train anywhere
[ ] Kettlebells          - Versatile training
[ ] Dumbbells            - Classic weights
[ ] Barbell & Rack       - Serious lifting
[ ] Pull-up Bar          - Upper body basics
[ ] Full Gym             - Everything available
```

MuscleMap will only suggest exercises you can actually do.

---

## Step 4: Start Your First Workout

Click **Start Workout** on your dashboard.

MuscleMap's prescription engine will generate a personalized routine based on:
- Your archetype
- Your equipment
- Your recovery status
- Available time

```
+------------------------------------------+
|            YOUR WORKOUT                   |
|------------------------------------------|
|  Exercise 1: Push-ups                     |
|  [MUSCLE MAP SHOWING CHEST/TRICEPS]       |
|                                           |
|  Target: 3 sets x 12 reps                 |
|  Rest: 60 seconds                         |
|                                           |
|  [Start Set]                              |
+------------------------------------------+
```

---

## Step 5: Log Your Sets

For each exercise:
1. Perform the movement
2. Enter your actual reps (and weight if applicable)
3. Rate your effort (RPE 1-10)
4. Rest and repeat

```
Set 1: [12] reps @ [--] kg  RPE: [7]  [Done]
Set 2: [11] reps @ [--] kg  RPE: [8]  [Done]
Set 3: [10] reps @ [--] kg  RPE: [9]  [Done]
```

## Common Questions

**Q: How many credits do workouts cost?**
A: 25 credits per workout. You start with 100 (4 free workouts).

**Q: Can I do my own exercises?**
A: Yes! You can add exercises during workouts or skip suggested ones.

**Q: What if I don't have time for a full workout?**
A: Set your available time before generating, or skip exercises as needed.

**Q: How does leveling work?**
A: Earn TU through workouts. Each level requires more TU than the last.
