# Global Leaderboard Plugin

Global leaderboards for Training Units and workout counts.

## Endpoints

### GET /api/plugins/leaderboard/tu/all-time
All-time TU leaderboard.

### GET /api/plugins/leaderboard/tu/weekly
Weekly TU leaderboard (last 7 days).

### GET /api/plugins/leaderboard/tu/monthly
Monthly TU leaderboard (last 30 days).

### GET /api/plugins/leaderboard/workouts?period=weekly|monthly|all-time
Workout count leaderboard.
